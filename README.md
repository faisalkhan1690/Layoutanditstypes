# Layout and its types
--------------
> A sample for how to use Android's Layout and its type 
This sample covers following things 


1. LinearLayout 
2. RelativeLayout 
3. FrameLayout


Pre-requisites
--------------

- Android SDK v23
- Android Build Tools v23.0.2

  
For more details follow doc
